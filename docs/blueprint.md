# **App Name**: Zentry Insights

## Core Features:

- Dashboard Overview: Display total calls, ongoing calls, dropped calls, AI/human handled ratio, and STT quality distribution.
- Call List: Browse and filter calls by date, status, and language, with pagination and call detail view access.
- Call Detail View: Display call metadata, transcript with STT confidence, AI processing timeline, guardrail modifications, and admin notes.
- Caller Overview: Anonymized view of caller behavior patterns, including call history and common intents.
- Admission Baseline View: Read-only display of program, quota type, estimated availability, confidence level, and last updated time.

## Style Guidelines:

- Primary color: Slate Blue (#708090) for a calm and professional tone.
- Background color: Off-White (#F8F8FF) to provide a clean, institutional feel.
- Accent color: Teal (#008080) to highlight important information and actions.
- Warning color: Orange/Amber (#FFB347) for low STT confidence or triggered guardrails.
- Failure color: Red (#F44336) for dropped calls or STT failures.
- Font: 'Roboto' (sans-serif) for a readable and modern look, suitable for transcripts and UI elements.
- Use simple, professional icons to represent call status, data types, and actions. Icons should be monochromatic in a shade of Teal.
- Employ a clear, modular layout with distinct sections for metadata, transcripts, and processing steps. Use subtle dividers to separate sections.
- Subtle transitions and animations for smooth navigation and feedback. Use a fade-in animation for loading new data.
